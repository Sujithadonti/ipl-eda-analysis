# 🏏 IPL Exploratory Data Analysis (2008–2020)

A data storytelling project analyzing 13 seasons of
IPL cricket using Python visualizations.

## 📌 Questions Explored
- Which team won the most matches?
- Which venues hosted the most games?
- Does winning the toss help win the match?
- How have matches per season changed over time?
- Bat first or field first — what do teams prefer?

## 📊 Key Findings
- Mumbai Indians have the most IPL wins
- Wankhede Stadium & Eden Gardens are top venues
- Winning the toss does NOT guarantee winning the match
- Teams prefer to field first after winning the toss

## 🛠️ Tech Stack
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Google Colab

## 📁 Dataset
- IPL Matches dataset (2008–2020)
- Source: Kaggle

## 🚀 How to Run
1. Download `matches.csv` from Kaggle
2. Upload to Google Colab
3. Run all cells in order
